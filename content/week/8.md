## Working in Pairs

For our second half today, we’ll demonstrate how you’ll work in code together—so it’d be helpful to sit with your partner! Our agenda:
<!-- .balance -->

1. In *one* of your GitHubs, start a [new repo](https://docs.github.com/en/get-started/quickstart/create-a-repo) named `spread` for our second projects—set to <samp>Public</samp>

1. Add your partner as [another user (*collaborator*)](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-access-to-your-personal-repositories/inviting-collaborators-to-a-personal-repository), which will let you both edit the repo’s code

1. Clone the repo locally, create an `index.html` with some basic HTML structure, commit it, and push ([as we did before](/week/3/#let-s-make-a-web-page))

1. Back on GitHub, [enable Pages](https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site) and link in the repo’s <samp>About</samp> section<br><br><br><span class="note">Switch drivers!</span><br><br>

1. Now on the *other* computer, clone the repo down from GitHub

1. Let’s make it responsive by adding our [viewport meta tag](/topic/responsive/#viewport-meta-tag)

1. Also add in [our reset](https://typography-interaction-2526.github.io/assets/reset.css) and create and link to a `style.css`

1. Start with some *mobile-first* styles! Use base [CSS variables](/topic/responsive/#briefly-css-variables) for colors, type size, spacing and a [media query](/topic/responsive/#media-queries) to make them responsive

1. Commit this and push it up—same as before!<br><br><br><span class="note">Switch drivers!</span><br><br>

1. Back on the *first* computer, pull down the changes

1. Try a bit of our new layout here—a [flex `column`](/topic/flexbox/) that switches to a `row` at larger breaks, variables for `gap`, etc.

1. Add a [`@media print` query](/topic/responsive/#screen-vs-print) and adjust color/<wbr>spacing properties for [the printed page](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_paged_media)

1. …repeat until the project is done!
