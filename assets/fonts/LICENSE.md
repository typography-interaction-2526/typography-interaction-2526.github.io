# *GortonDigital*

`gorton-digital--heavy.otf` \
`gorton-digital--light.otf` \
`gorton-digital--medium.otf` \
`gorton-digital--regular.otf`

These are based on [the revival work](https://github.com/drdnar/GortonDigital), done by [N. E. Bartgis](https://github.com/drdnar):

> These typefaces are free to use for non-commercial purposes.
>
> However, if you want to use these typefaces for a commercial purpose, you will need a license. It is not my intention to make any significant money from this, but, at Richard Gorton's suggestion, we will select a US-based educational non-profit for you to donate your licensing fee to. Contact me at rsbartgis@gmail.com for details.

I’ve [forked the repo](https://github.com/mfehrenbach/GortonDigital) and have opened [a PR with additions](https://github.com/drdnar/GortonDigital/pull/8); the licensing would remain the same.

# *OCR-B*

## Original

`ocr-b--original.otf`

Renamed from [the Tsukurimashou Project work](https://tsukurimashou.org/ocr.php.en), done by [Matthew Skala](https://ansuz.sooke.bc.ca):

> I translated existing Metafont sources to MetaType1 to create the enhanced TrueType, PostScript, and OpenType versions included in this package. They are available for free use, including in commercial applications; my own contributions are public domain, and others have licensed their copyright claims.
